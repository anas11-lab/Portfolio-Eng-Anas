# Anas Awadh | Portfolio

A professional portfolio website showcasing software engineering projects, skills, and experience.

## Features

- **Modern UI/UX**: Built with React and Tailwind CSS for a sleek, responsive design.
- **Interactive Projects**: Showcase of full-stack, mobile, and specialized software solutions.
- **Smooth Animations**: High-performance animations using Framer Motion (motion/react).
- **Multilingual Support**: Available in both English and Arabic.
- **Dynamic Content**: Easy to update projects, experience, and certificates.

## Tech Stack

- **Frontend**: React 18+, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

### Building for Production

Create an optimized production build:
```bash
npm build
```

## Deployment

### GitHub Pages (Recommended)

1. Push this code to your GitHub repository.
2. Go to your repository **Settings** > **Pages**.
3. Under **Build and deployment** > **Source**, select **GitHub Actions**.
4. The site will automatically deploy whenever you push to the `main` branch.

### Manual Deployment

The project is ready to be deployed on any static hosting platform such as Vercel or Netlify.
